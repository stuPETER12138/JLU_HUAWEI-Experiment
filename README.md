# JLU_HUAWEI-Experiment
吉林大学电机拖动课程华为实验10组 智能小车的相关代码

简要概述：

Hardware ： 适配该智能小车及相关外设的代码，具有驱动，导航，避障的相关功能。

Software ： 智能小车的目标检测代码，实现识别目标物体，并导航前往的功能。

Model    ： 本小车用到的目标检测的模型。
